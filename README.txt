dir - файловый менеджер
tab - переход из левого окна в правое и наоборот
backspace - выход
стрелочки вверх/вниз - навигация
enter - открыть каталог
с - скопировать выбранный файл в каталог, открытый в соседнем окне
обозначения в окнах
белый фон - выделенный файл/каталог
синий текст на черном фоне - каталог
белый текст на черном фоне - файл
зеленый текст - неактивное окно
